  SCRIPTY MOD (Forge)

=========
Setting up the workspace: run "gradlew setupDecompWorkspace"
Building the mod: run "gradlew build"
Running the client/server with the mod automatically installed: run "gradlew runClient" and "gradlew runServer"
=========

CI builds can be found here: http://bamboo.gserv.me/browse/SCRPTY-MOD
